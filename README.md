## 仕様

ライントレースロボットは、3つのフォトリフレクタを利用して黒線を検知します。
フォトリフレクタから得られるアナログ値を基にPID制御を行いライントレースを行います。

ロボットが黒線に対してどちら側にいるか常に把握している為、コースアウトしてもすぐに復帰することが可能です。

追従ロボットは左右の超音波センサの値の差を基に追従を行います。
